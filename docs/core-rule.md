# **7팀 꼬리잡기 코어전 - 상세 게임 진행 방식**

## **1. 게임 개요**

- 게임 형태: 7팀 코어 파괴전 + 꼬리잡기 타겟 인수인계
- 팀: Red, Orange, Yellow, Green, Blue, Indigo, Purple
- 승리 조건: 마지막까지 생존한 1팀
- 핵심 룰: 내 팀의 **현재 타겟 코어**만 공격 가능

## **2. 기본 설정값**

- 코어 HP: 500
- 코어 데미지 가능 무기: 다이아몬드 검
- 빌드 페이즈: 10분 (600초)
- 결계 반지름: 320
- 코어 보호 반경: 4블록
- 종료 후 전원 텔레포트: 5초 후

## **3. 게임 시작 전 운영 준비**

### **3.1 맵 준비**

- 원형 결계 맵 구성 (중심 (0,0), 반지름 320)
- 중앙 상점 구역 배치
- 7개 베이스캠프 배치
- 각 베이스 내부 코어 챔버(코어 방) 구성

### **3.2 코어 좌표 확인**

현재 적용 좌표(world 기준):

- Red: (100, 65, 200)
- Orange: (220, 65, 120)
- Yellow: (220, 65, -40)
- Green: (100, 65, -120)
- Blue: (-20, 65, -120)
- Indigo: (-140, 65, -40)
- Purple: (-140, 65, 120)

필요 시 운영자가 /setcore <team> 으로 실좌표 재지정.

### **3.3 팀 배정**

- 운영자가 /setteam <offlineplayer> <team>으로 배정
- 내부 팀 변수 + LuckPerms 그룹 자동 동기화

## **4. 게임 진행 단계**

### **4.1 단계 A - 게임 시작**

- 운영자: /coregame start
- 서버 자동 처리:
    - 팀/팀색 자동 보정
    - 그룹 생성 보정
    - 코어 HP/타겟/생존 상태 초기화
    - 페이즈를 build로 설정
    - PvP OFF

### **4.2 단계 B - 빌드 페이즈 (10분)**

- 플레이어는 기지 건축, 방어선 구축, 자원 정비 가능
- 코어 공격 불가
- PvP 불가
- 결계 밖 설치/파괴 불가
- 코어 보호 반경 내 설치/파괴 불가

### **4.3 단계 C - 전투 페이즈 시작**

- 10분 경과 시 자동 전환
- 페이즈 combat으로 변경
- PvP ON
- 코어 공격 ON

### **4.4 단계 D - 코어 공략/탈락 처리**

코어 타격 판정 순서:

1. 코어 블록인지 확인
2. 공격자 팀 확인
3. 공격자 현재 타겟 팀 확인
4. 타격한 코어가 타겟 팀 코어인지 판정

결과 분기:

- 타겟 코어 + 다이아몬드 검: 코어 HP -1
- 타겟 코어지만 무기 불일치: 데미지 없음
- 자기 코어: 차단
- 타겟 외 코어: 차단

코어 HP가 0이 되면:

- 해당 팀 탈락
- 해당 팀 전원 관전 모드
- 공격팀이 탈락팀의 타겟을 인수

### **4.5 단계 E - 승리/종료**

- 생존 팀 수가 1팀이 되는 즉시:
    - 승리팀 공지
    - 게임 상태 종료
    - PvP OFF
    - 5초 대기 후 전원 종료 지점 텔레포트

## **5. 타겟 체인 규칙**

초기 체인:

- Red -> Orange -> Yellow -> Green -> Blue -> Indigo -> Purple -> Red

인수인계 규칙:

- A팀이 B팀 코어를 파괴하면,
- A팀의 새 타겟은 기존 B팀의 타겟
- 이미 탈락한 팀이면 다음 생존팀으로 스킵

## **6. 베이스캠프 구성 표준**

각 팀 베이스 권장 구성:

- 코어 챔버: 코어 1개 + 보호 반경 내 핵심 구조
- 스폰 구역: 리스폰 및 집결
- 방어 라인: 주 진입로/보조 진입로 방어 구조
- 보급 구역: 자재/장비 정비 구간

맵 컨셉(스타크래프트 느낌):

- 중앙 교전 유도
- 베이스 간 초크포인트
- 고지/저지 혼합 지형
- 외곽 우회 동선 제공

## **7. 운영 명령어**

- /coregame start
- /coregame stop
- /coregame status
- /setteam <offlineplayer> <team>
- /setcore <team>

## **8. 체크리스트 (운영자용)**

- [ ]  7팀 인원 배정 완료

- [ ]  결계/상점/베이스 점검 완료

게임 중:

- [ ]  코어 HP 감소/차단 메시지 정상 확인
- [ ]  탈락 시 타겟 인수인계 정상 확인

게임 종료 시:

- [ ]  승리 공지 확인
- [ ]  5초 후 전원 텔레포트 확인
